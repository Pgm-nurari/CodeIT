Section 01

Anchor Tag

Here the anchor tag, <a href=""> </a> is used to create a link between various files throughout the internet or in the same server.
We can also use the download attribute in anchor tag to make a linked file downloadable into the user's pc

Link Tag

Here the link tag, <link rel="" href="" type=""> is used to create a link between various files throughout the server, but it can only be used for head and not in the body tag.

Nav Tag

The <nav> </nav> is used to navigate through the various sections in the same HTML file.
Anchor tag can be used to create the hyperlink for the section, with the href="#section_id" attribute in it. While giving simply "#" in href will take us back to the top of the file.
